So funktioniert's

In 3 Schritten zur Traumwohnung

Einmal einrichten, dann läuft alles von alleine.

1

App herunterladen & installieren

Lade WohnungFinder herunter und installiere die App. Beim ersten Start — Mac: Rechtsklick → Öffnen → Öffnen. Windows: „Weitere Informationen“ → „Trotzdem ausführen“. Sicherheitsabfrage einmalig bestätigen.

2

Filter & Nachricht konfigurieren

Stelle deine Suchfilter ein, melde dich mit deinem ImmoScout24-Konto an und schreib deine Nachrichtenvorlage.

3

Automation starten

Klicke auf "Automation starten" oder aktiviere den Zeitplan – WohnungFinder sendet automatisch Anfragen an passende Inserate.
